# SQL Antipatterns

## 물리적 데이터베이스 설계 안티패턴

### 반올림 오류

- 목표: 정수 대신 소수 사용
  - float 칼럼을 추가했지만, 소수계산이 안 맞음
  - 소수 둘째 자리까지 표현하자
- 안티패턴: FLOAT데이터 타입 사용
  - 필요에 의한 반올림
  - SQL에서 FLOAT 사용
- 안티패턴 인식 방법
  - FLOAT, REAL, DOUBLE PRECISION 데이터 타입 사용
- 안티패턴 사용이 합당한 경우
  - INTEGER나 NUMBER 타입의 지원범위 보다 큰 범위의 실수 사용
- 해법: NUMERIC 데이터 타입 사용
  - 고정 소수점 수에는 NUMERIC 또는 DECIMAL 타입 사용
