### 연결리스트
- 현재 데이터와 다음 데이터 주소를 저장
    | a    | 1482 | ... | b    | 1997 | ... | c    | null |
    |------|------|-----|------|------|-----|------|------|
    | 1001 | 1002 |     | 1482 | 1483 |     | 1997 | 1998 |

- 연결 리스트 구현
    ```c
    // 간단한 예시
    node1 = Node.new("once")
    node2 = Node.new("upon")
    node1.nextNode = node2

    node3 = Node.new("a")
    node2.nextNode = node3

    node4 = Node.new("time")
    node3.nextNode = node4
    ```
